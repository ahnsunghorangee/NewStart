# Double Exclamation

느낌표 2개 연산자는 확실한 논리 결과(참 또는 거짓)를 얻기 위해 사용하는 연산자이다.



기존의 느낌표 연산자는 true를 false, flase를 true로 반전하는 기능을 가지고 있는데,



느낌표 2개 연산은 반전에 반전을 거듭하는 형태이므로 undefined나 null값을 false로 불리언 형변환하는 데에 사용할 수 있다.



```js
var case1;  // undefined

console.log("case1  : " + (case1)); // undefined
console.log("!case1 : " + (!case1)); // true
console.log("!!case1: " + (!!case1)); // false
console.log('') // 개행

var case2 = true; // 불리언 데이터 

console.log("case2  : " + (case2)); // true
console.log("!case2 : " + (!case2)); // false
console.log("!!case2: " + (!!case2)); // true
console.log('') // 개행

var case3 = null; // null 

console.log("case3  : " + (case3)); // null
console.log("!case3 : " + (!case3)); // true
console.log("!!case3: " + (!!case3)); // false
```



결과에서와 같이 느낌표 2개 연산자는 null이나 undefined값을 false로 변환할 때 사용할 수 있다.
